# Editor de Texto

[Visual Studio Code](https://code.visualstudio.com/)

## Settings

```sh
{
    "terminal.integrated.shell.windows": "C:\\WINDOWS\\System32\\wsl.exe"
}
```

## Dependências

```sh
npm install -g eslint
npm install -g prettier
```
